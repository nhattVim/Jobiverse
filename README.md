## Điều kiện

-   Cài [nodejs](https://nodejs.org/en)

-   Cài yarn `npm install -g yarn`

## Cách dùng

-   clone dự về máy

    ```
    git clone https://github.com/nhattVim/Jobiverse.git
    ```

-   backend

    ```
    cd Jobiverse/backend
    yarn dev
    ```

-   frontend

    ```
    cd Jobiverse/frontend
    yarn dev
    ```
